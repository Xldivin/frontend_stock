# Stock_Drop_fe

Starter frontend using React + TypeScript + Vite.

## Run

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

Default API base URL is `http://localhost:4000` and can be overridden via `VITE_API_BASE_URL`.
